# Cafecito Tech Landing Page (V2) — Brief de Cambios + Copy Cleanup
**Owner:** Link Foundation / Cafecito Tech  
**Solicitante:** William Suarez  
**Fecha:** 2026-01-26

> Este brief organiza los cambios solicitados para el landing page de **Cafecito Tech**.  
> Está diseñado para que un A&E / Gen / dev lo pueda ejecutar de forma clara y rápida.

---

## 1) North Star (Intención del Landing)

### 1.1 Qué queremos lograr
- Que el landing se sienta **casual, cálido, humano** — como una comunidad creada entre amigos.
- Que NO se sienta corporativo.
- Que quede **clarísimo** que Cafecito Tech es una iniciativa de **Link Foundation**.
- Que se sienta como una página “dentro” del ecosistema Link: **navbar + footer + estructura**.

### 1.2 Sensación / Vibe
**Debe sentirse como:**
- “Un cafecito entre amigos”
- Comunidad latina en Charlotte curiosa por tecnología
- Espacio para aprender, conectar y navegar la revolución tecnológica juntos

**Evitar:**
- Lenguaje demasiado formal
- Exceso de texto largo
- Repetición constante de frases (ej. “ven a conversar”)

---

## 2) Problemas actuales a solucionar

### 2.1 Copy repetitivo / redundante
- Se repiten ideas similares muchas veces (especialmente “ven a conversar”).
- Hay frases que dicen lo mismo con diferentes palabras.
- No está optimizado para lectura rápida (escaneo).

✅ Objetivo: **hacerlo más claro, corto, fácil de leer y más explicativo** para alguien que llega por primera vez.

### 2.2 Link Foundation no está lo suficientemente visible
- Queremos más presencia de Link como “marca madre”
- Manteniendo el estilo visual de Cafecito Tech (warm community vibe)

### 2.3 Registro actual con Luma (se cambia)
- Se eliminará Luma como sistema de registro.
- Se reemplazará por **GoHighLevel** con event pages por cada evento.

---

- **A veces**: invitados expertos o partners (en español)

### 4.4 Sección: “Temas que vamos a tocar”

## 8) Copy Review: Revisión y limpieza del texto

### 8.1 Objetivo del copy final
- Menos repetición
- Más claridad
- Más escaneable (lectura rápida)
- Más explicativo para alguien nuevo
- Con Link Foundation más claro
- Con espíritu “amigos que lo crearon”

---

### 8.2 Reglas de edición del copy (para el redactor/editor)
✅ Mantener:
- Energía cálida
- Mensaje “momento de revolución”
- Comunidad latina tech Charlotte

✅ Evitar:
- Repetir “ven a conversar” más de 1 vez
- Repetir “comunidad” cada párrafo
- Bloques de texto muy largos

✅ Estructura recomendada:
- Títulos cortos
- 1–2 líneas por párrafo
- Bullets para “qué incluye”

---

### 8.3 Mensajes obligatorios (deben quedar explícitos)
1) **Relación con Link Foundation**
   - “Cafecito Tech es una iniciativa de Link Foundation.”

2) **Origen humano**
   - “Lo creamos como amigos, porque queríamos un espacio así en Charlotte.”

3) **Qué pasa en el evento**
   - 30 min cafecito + networking
   - tema del mes + discusión
   - algunos eventos tendrán workshops y demos
   - invitados y partners (en español)
